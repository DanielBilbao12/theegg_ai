# ENUNCIADO DE LA TAREA 43

Un cracker ha accedido a una información privada y tú, que eres "El inspector SQL", debe tomar una serie de decisiones mediante sentencias SQL para resolver el caso. Escribe en un documento de texto (.doc, .txt, ...) la sentencia SQL que has utilizado para cada caso resuelto junto con una breve explicación sobre la sentencia que has utilizado.

# EXPLICACIÓN

Para la resolucion de esta tarea se ha seguido la propia guia que proporciona la web en la que se alojan los diferentes casos a resolver: [Enlace](https://sqlpd.com/)

Cabe destacar que, al resolver varios casos (entorno a 10), el "tutorial" se vuelve de pago. Al terminar el tutorial, si se refresca la página, es posible volver a realizar el tutorial pero los casos a resolver son practicamente identicos.

Tras realizar varias veces el tutorial, se ha optado por realizarlo por última vez sin mirar la guia y documentar las sentencias SQL utilizadas.
